How the workshop runs.

EVERY UNIT · EVERY SHIP

Written down firstQC · PASS

Plans and specs are committed to the repository before the code that implements them. When a build deviates from its plan, the deviation is recorded inline — and the blog documents what actually happened afterwards, including the mistakes and the dead ends.

Tested before deployQC · PASS

Every push runs a browser-level smoke suite across every page of this site — 266 checks as of July 2026, from checkout flows to WebGL boot markers. If a single check fails, nothing goes live. The suite count on this page is updated when the suite changes.

Honest by defaultQC · PASS

Prices are exact, limits are stated, and "coming soon" means a disabled button — not a mailing-list trap. Copy is held to the same tolerance as the code: if a feature isn't finished, the page says so instead of pretending it exists.
